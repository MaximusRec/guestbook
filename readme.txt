Загрузка сообщений гостевой книги + изображения и всё без перегрузки страницы.
Стек техноллогий: PHP, JavaScript, jQuery, Ajax

guestBook.php - основной класс с методами
index.php - основной файл с интерфейсом

read.php - выводо оставленных комментов с пагинацией
show.php - показ еденичной карточки
upload.php - логика загрузки и "ресайза" фотографий
